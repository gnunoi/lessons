# B2028反向输出一个三位数

## 题目描述

将一个三位数反向输出，例如输入 $358$，反向输出 $853$。

## 输入格式

一个三位数 $n$。

## 输出格式

反向输出 $n$。

## 输入输出样例 #1

### 输入 #1

```
100
```

### 输出 #1

```
001
```

## 输入输出样例 #2

### 输入 #2

```
001
```

### 输出 #2

```
100
```

## 输入输出样例 #3

### 输入 #3

```
678
```

### 输出 #3

```
876
```

## 代码

```cpp
#include<iostream>
using namespace std;

int main()
{
    int a, b, c, d;
    cin >> a;

    b = a % 10; // 个位数
    c = (a % 100 - b) / 10; // 十位数
    d = a / 100; // 百位数

    cout << b << c << d;
    
    return 0;
}
```
