# B2040判断是否为两位数

## 题目描述

判断一个正整数是否是两位数（即大于等于 $10$ 且小于等于 $99$）。

## 输入格式

一个正整数，不超过 $1000$。

## 输出格式

一行。若该正整数是两位数，输出 $1$，否则输出 $0$。

## 输入输出样例 #1

### 输入 #1

```
54
```

### 输出 #1

```
1
```

## 代码

```cpp
#include <iostream>
using namespace std;

int main()
{
    int a;
    
    cin >> a;

    if (a >= 10 && a <= 99)
        cout << 1 << endl;
    else
        cout << 0 << endl;

    return 0;
}
```
