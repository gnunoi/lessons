# B2062 乘方计算

## 题目描述

给出一个整数 $a$ 和一个正整数 $n$，求乘方 $a^n$。

## 输入格式

一行，包含两个整数 $a$ 和 $n$。$-1000000 \le a \le 1000000$，$1 \le n \le 10000$。

## 输出格式

一个整数，即乘方结果。题目保证最终结果的绝对值不超过 $1000000$。

## 输入输出样例 #1

### 输入 #1

```
2 3
```

### 输出 #1

```
8
```

## 代码

```cpp
// B2062 乘方计算
// 给出一个整数a和一个正整数n，求乘方a^n

#include<iostream>
using namespace std;

int main()
{
    int a, n, p = 1;
    cin >> a >> n;
    for(int i = 0; i < n; i++) p *= a;
    cout << p << endl;
    return 0;
}
```
