# U549458 求两个正整数的最小公倍数。

## 题目描述

1. 定义一个函数int gcd(int a, int b)，求两个正整数的最大公约数。
1. 再定义一个函数int lcd(int a, int b)，求两个正整数的最小公倍数。

## 输入格式

输入两个正整数$a$，$b$，均满足：$1≤a,b<10^4$。

## 输出格式

输出a,b的最小公倍数。

## 输入输出样例 #1

### 输入 #1

```
2 3
```

### 输出 #1

```
6
```

## 输入输出样例 #2

### 输入 #2

```
4 6
```

### 输出 #2

```
12
```

# 源代码

```cpp
#include <iostream>
using namespace std;

// 函数声明
int gcd(int a, int b);
int lcm(int a, int b);

int main() 
{
    int a, b;
    
    cin >> a >> b;
    
    int result = lcm(a, b);
    cout << result << endl;
    
    return 0;
}

// 计算最大公约数的函数
int gcd(int a, int b) 
{
    while (b != 0) {
        int temp = b;
        b = a % b;
        a = temp;
    }
    return a;
}

// 计算最小公倍数的函数
int lcm(int a, int b) {
    return (a / gcd(a, b)) * b;
}
```
